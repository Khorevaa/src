========================================================================
       CONSOLE APPLICATION : V8Unpack - доработка
========================================================================

Командная строка для запуска

	V8Unpack -U[NPACK]  in_filename.cf   out_dirname
	V8Unpack -PA[CK]    in_dirname       out_filename.cf
	V8Unpack -I[NFLATE] in_filename.data out_filename
	V8Unpack -D[EFLATE] in_filename      filename.data

	V8Unpack -P[ARSE]   in_filename      out_dirname  - распаковываем конфигурацию, обработки и т.д.
	V8Unpack -B[UILD]   in_dirname       out_filename - сборка конфигурации, обоработки и т.д.
